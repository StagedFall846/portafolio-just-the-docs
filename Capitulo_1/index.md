# Capítulo 1: Redes Neuronales

En este apartado se presentan los fundamentos y las implementaciones prácticas relacionadas con las redes neuronales artificiales. Exploraremos desde modelos básicos como el perceptrón simple hasta arquitecturas más complejas.

## Contenido del Capítulo
A continuación, encontrarás los ejercicios desarrollados en clase:

1. **Perceptrón Simple:** Clasificación de funciones lógicas (OR, AND).
2. **Brazo Robótico (2 GDL):** Aplicación de redes neuronales para cinemática.
3. **Entrenamiento y Validación:** Análisis de error y ajuste de pesos sinápticos.

Utiliza el menú lateral para navegar por los ejercicios específicos.
